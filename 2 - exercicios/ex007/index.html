Principais Formatações


1. Negrito/Destaque

A tag usada para negrito é </b>, (sem semântica, usada na CSS).

Para ter signficado usa-se a tag </strong> que deixa o período selecionado destacado.

2. Itálico/Ênfase

A tag utilizada para a forma itálico é a tag </i>, (sem semântica, usada na CSS).

Para dar ênfase em um período selecionado, usa-se a tag </em>, da qual contém significado no HTML5.

3. Texto Marcado

Para destacar na forma como um marcador usa-se a tag </mark>

3.1. Mudar cor do marcador

Para mudar a cor do marcador de uma palavra marcada, usa-se o paramentro style dentro da tag mark.

OBS: USANDO ESTE PARAMENTRO DENTRO DA TAG EM SI,
NÃO SERÁ APLICADO A COR EM OUTRAS MARCAÇÕES.
DEVERÁ ADD A TAG STYLE NO HEAD DO CÓDIGO JUNTO A UM SELETOR.

4. Texto Grande e pequeno

Para usar um texto ou palavra grande, usa-se a tag </big>, porém ela é obsoleta, então use a CSS.

Agora para um texto ou palavra pequena, usa-se a tag </small>.

5. Texto deletado

Eis o exemplo de uso do Texto Deletado, usamos a tag </del>

6. Texto inserido(sublinhado)

Um Texto Inserido é o mesmo que Texto Sublinhado, mas com semântica

Usa-se as tags </ins> e </u>, respectivamente.

7. Texto Sobrescrito e Subscrito

Exemplo de texto sem a aplicação das tags -> Fórmula da água: H2O - Fórmula matemática: 2x2+3

Exemplo de texto com aplicação das tags </sup> e </sub>:

Fórmula matemática: 2x2+3

Fórmula da Água: H2O
